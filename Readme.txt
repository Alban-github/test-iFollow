Présentaton des réponses et du travail dans le fichier Readme :

Pour chaque exercice, lancer le fichier correspondant (exercice1.py, 2, 3, etc).
Cela ouvrira plusieurs terminaux quand besoin.

Avant de lancer le fichier suivant :
fermer tout ce qui a été ouvert par le fichier pricipal de l'exercie
ou mettre une partie du code en commentaire, par exemple le lancement de roscore/gazebo qui doit être initialisé

Des explications sont présentes en commentaire dans les fichiers code source.
Certains fichiers font appel à des fonctions définies dans les autres.
Il faut aussi avoir installé les prérequis de ROS, example Ubuntu 20.04, ROS noetic (20.04), openCV, et d'autres lib.

Dans mon cas j'ai un windows 10 donc j'ai du installer ubuntu x64 sur VMware et tous les packages à chaque étape.
Utiliser une machine virtuelle a posé beaucoup de problèmes, en particulier car des fonctionalités basiques ne fonctionnaient pas correctement. J'ai donc été forcé de trouver des solutions d'installation à chaque fois au lieu de me concentrer sur la partie programation python.
